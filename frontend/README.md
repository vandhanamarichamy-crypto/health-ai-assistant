# 🩺 Health AI Assistant

An AI-powered HealthTech web application that analyzes lifestyle habits and provides personalized wellness insights.

## 🚀 Features

- Sleep tracking
- Water intake monitoring
- Stress analysis
- Dynamic health risk detection
- AI-generated wellness suggestions
- Modern responsive UI

## 🛠️ Tech Stack

### Frontend
- React
- Tailwind CSS
- Vite

### Backend (Upcoming)
- FastAPI
- Python
- Scikit-learn

### AI Integration (Upcoming)
- Gemini API

## 📂 Project Setup

```bash
git clone https://github.com/vandhanamarichamy-crypto/health-ai-assistant.git
```

```bash
cd frontend
npm install
npm run dev
```
